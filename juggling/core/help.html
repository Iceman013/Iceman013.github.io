Help

Menu

The menu is accessed with the "Menu" button on the top left. Once selected, it will open a sidebar with the main navigational pages. Simply click any of these pages to go there.

Trick List

The trick list, accessed via the "Tricks" button in the menu, has the complete list of tricks available on this site. Near the top of the page is a set of checkboxes. These are filters. In order to narrow your search, check any filters you want. The list will then consist of only the tricks that meet all the rules you set. Uncheck any filter to deactivate it. Each box is a link to a trick. Click on it to go to that page. It has a title, the title of the trick. Then there are tags that can be filtered. Beneath that is a brief description of the trick. Below that is the list of recommended prerequisite tricks.
