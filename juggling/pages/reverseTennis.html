Reverse Tennis

Description

This trick is a classic next step implied by the tennis trick. It uses the same idea but is swapped.

Prerequisites

These are recommended tricks to learn before you do this one. It may help.
  • Tennis

Tutorial

This trick starts with reverse cascade. Then, on every third throw, perform an inside cascade throw. This should look like two balls going in big arcs and one in little arcs in the middle.

Tips

  • This is just tennis with cascade and reverse cascade swapped.
  • The tricks used in tennis work here as well. Using a different colored ball and making more exagerated motions helps.
