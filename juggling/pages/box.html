Box

Description

This trick is a classic three ball pattern that resembles the sides and bottom of a box.

Prerequisites

    Tutorial

    Step 1

    This step uses only 2 balls, one in each hand. It starts with a simultanious vertical throw and a sideways pass. This needs to be practiced on both sides.

    Step 2

    Start with one throw in the air. Then, begin the pattern. You'll notice that the 3 balls form a type of L. This rotates back and forth 90 degrees. Each ball has 2 states it goes in a line between. Thinking of it in this way can make it easier to understand.

    Tips

    • This trick becomes more impressive as it gets cleaner. At first, your pattern will be sloppy. This will take some practice to make it look good.
    • This trick has many many variations. It can be used very effectively to change directions during a shower.
