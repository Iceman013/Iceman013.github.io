Crossovers

Description

This is a variation of the columns trick that involves crossing balls.

Prerequisites

These are recommended tricks to learn before you do this one. It may help.
  • Columns

Tutorial

You will notice the similarities between this trick and columns. Both use the same concept except crossovers cross the outside balls in the air. This makes it slightly more challenging but it looks more impressive because balls change position.

Tips

  • You may have difficulty with the crossing balls colliding. Varying how each hand throws can help prevent this.
